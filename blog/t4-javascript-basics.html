20.03.2016

How does Javascript compare to HTML and CSS?

HTML is the structure of a webpage. CSS is the style of a webpage, and javascript allows the webpage to be interactive and perform actions.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'

Control flow is the order that code is executed , and a loop is a piece of code that repeats while a condition is true or until a condition is met.
For example, everyday, the first piece of 'code' that is executed 'wake up'. This must occur before any other code can be executed>, and because I wake up everyday, and I don't want to repeat this code for the number of days i plan on waking up, I can loop it. The loop would continue while a condition was true. In this example, the condition might be to keep looping while I less than 100 years old, any older and the loop will cease to continue looping.

Explain the difference between accessing data from arrays and object literals

Arrays store values according to an index, and and objects store values according to corresponding keys (aka properties).
To access data in an array you can specify the index at which the value is stored (ie the first value, second value ..etc), but to access a value of an object you must specify the property that the value is paired with(ie the value of the age property, name property, colour property ..etc)

Explain what functions are and why they are useful

Functions are pieces of code that perform a specific task. They are useful for performing tasks that may need to be done many times, instead of writing out the same piece of code over and over in different places.
