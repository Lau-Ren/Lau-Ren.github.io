Javascript Language

18.04.2016

In the JavaScript language, describe all the use cases for each of:

parentheses ()

execute functions (e.g. functionName(), console.log()),
express conditons (e.g. if () {} else if () {}, for () {}, while () {})

square brackets []

create an array (e.g. var arr = []),
point to an index in an array (e.g. arr[i]),
point to a key in an obeject (e.g. house.paint["colour"])

braces {}

enclose conditional statements (e.g. if (...) {x+y} else if (...) {x-y}),
function definitions (e.g. function myFunc() {definition here}),
loops (e.g. for () { loop thorugh the things in here}),
or declare objects (e.g. var myObj = {"key": "value"})

single ' '/double quotes " "

enclose strings (e.g. var str = "Im a string"), enclose object keys in object (e.g. {'key': 'value'})
Single and double quotes can be used interchangeably. However, single quotes must be used inside double quotes and vice versa (e.g. console.log('say "hello"') or console.log("say 'hello'"))
