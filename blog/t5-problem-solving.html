Problem Solving

27.03.2016

In programming I often find that the most difficult part of solving a problem is getting started slving it. I find that by accepting that the first attempt will probably not work is a good way of breaking down this barrier. Once i have something to work with then i can go about figuring out where or why it wont work.

Reflect on how confident you feel using the problem solving techniques and process:

Psuedocode. I have used this method often, it helps me get the logic of a solution straight in my head

Trying something. This is my first line of defence when solving problems.

Rubber ducky method. i dont use a rubber ducky, but i find that explain my problem to another person often helps. there have been anumber of times where by the time i have finished saying my problem outload I have already figured out the solution.

Reading error messages. I dont feel confident with error messages, but i have found them useful in identifying the location of the problem

Console.logging. I use this method before i even know i have a problem. When completing the racer assignment, i was console.logging my variables just to keep track of their values and make sure they were what they were supposed to be.

Googling. Very familiar with this method, and I feel condident using it.

Asking your peers for help I prefer to use other methods before asking others because i feel like the more i struggle the more i am learning!

Asking coaches for help. I am also less inclined to ask coaches for help, for the same reason as above. Although, im sure in some cases it would be more time effective to have some one point me in the right direction for a solution

Improving your process with reflection. I have found reflection to be a useful tool in the learning process, but I havent used it as a way of improving my problem solving process.

Describe what these functions do in your own words:

.map() this method takes an array of already existing elements, and applies a function to each of them. These new values are then mapped out into a new array. For example, this method could take an array of numbers 1-9 and times each valueby ten, and map out the new values 10-90 to the new array

.filter()this method allows you to filter out values in an array that meet a certain criteria and make a new array that excludes thse values that do not meet the criteria. For example, values in an array that a smaller than 5 could be filtered out and all numbers over 5 would form the new array.

.reduce()This method allows you to take an array of multiple values and reduce it to a single values. For example, an array of strings could be reduced to one long string, and array of numbers could be reduced to the sum of those values...etc
